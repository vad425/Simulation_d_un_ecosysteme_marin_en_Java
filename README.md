# Simulation_d_un_ecosysteme_marin_en_Java
Projet de simulation orientée objet d'un écosystème marin développé en Java.  Un terrain en grille accueille différentes ressources marines (poissons herbivores, carnivores, invasifs, crustacés, algues, huîtres) qui évoluent de manière autonome à chaque tour de simulation. Un agent (bateau de pêche) se déplace sur le terrain, pêche des ressources et gère un solde en fonction de ses prises.  Concepts POO mis en œuvre : - Héritage et classes abstraites (Ressource, RessourcesMouvements, RessourcesStatiques) - Interfaces (InterfaceVivant, InterfaceMobile, InterfaceStatique) - Constructeur par copie (Agent) - Gestion des exceptions personnalisées (GestionException) - Collections (ArrayList) - Javadoc  Auteur : MALISEV Vadim
